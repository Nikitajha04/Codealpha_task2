# JavaScript Calculator Project

A basic yet responsive Calculator built using **HTML**, **CSS**, and **JavaScript**.  
This project performs essential arithmetic operations and includes keyboard input support. It showcases front-end development skills such as event handling, logical operations, and responsive UI design.

---

## 🔹 Features

- Supports addition, subtraction, multiplication, and division  
- Clear screen and delete functionality  
- Real-time result display  
- Keyboard input support  
- Responsive design using CSS Grid and Flexbox  
- Clean, modern, and minimal user interface  

---

## 🛠️ Technologies Used

- **HTML5** – Application structure  
- **CSS3** – Styling, layout, and responsive design  
- **JavaScript (ES6)** – Core logic and user interaction
